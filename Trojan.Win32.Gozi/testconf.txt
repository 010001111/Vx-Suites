Тестовый конфиг-файл, описание


Вкладка "Replace"

Подмена части контента в документе (HTTPs)
Страница: click.alfabank.ru
Все слова "Альфа" в тексте документа заменяются на "Бета"

Инжект скрипта в документ (HTTPs, SPDY)
Страница: https://accounts.google.com
При открытии выдаёт диалоговое окно с текстом "Hello SPDY!"

Инжект скрипта в документ (HTTPs)
Cтраница: https://web.de
При открытии выдаёт диалоговое окно с надписью "Hello HTTPS!"


Вкладка "Full replace"

Подмена страницы (HTTP)
Страница: gismeteo.ru
Кликнуть на "Санкт-Петербург", будет открыта страница для "Алматы"

Подмена страницы (HTTPs, SPDY)
Страница: twitter.com/barackobama
Будет выведен контент страницы: twitter.com/sashagrey


Вкладка "Grab"

Отправка фрагмента контента на сервер (HTTP, HTTPs)
Страница: vk.com/dm
При открытии страницы, текст с её описанием будет отправлен не сервер

Сохранение фрагмента страницы в переменную (HTTP, HTTPs)
Страница: vk.com
Ввести имя пользователя и пароль. Имя пользоватля будет отправлено на сервер и сохранено
 в переменную UserName для дальнейшего использования.
Переменные хранятся в реестре: HKEY_CURRENT_USER\Software\AppDataLow\[GUID]\Vars 


Вкладка "Screenshot"

Отправка снимка экрана по ключевому слову (HTTP)
Страница: wikipedia.org
В строке поиска набрать "john lennon" и нажать Enter.
Будет сделан и отправлен на сервер скриншот открывшейся страницы.


Вкладка "Scan files"

При получении конфига будет произведен поиск файлов с именем "hosts". Найденые файлы будут отправлены на сервер.


Вкладка "Post Params"

Сохранение имени пользователя при отправке POST-запроса (HTTPs)
Страница: mail.yahoo.com
Ввести имя пользователя и пароль. Имя пользователя будет сохранено в переменную "acount", пароль в переменную "pass".
Переменные хранятся в реестре: HKEY_CURRENT_USER\Software\AppDataLow\[GUID]\Vars
В дальнейшем, указанные переменные можно использовать в условиях, задваемых в конфиге. См. вкладку "Get URL" ниже. 


Вкладка "Hidden"

Блокировка загрузки страницы (HTTP)
Страница: kaspersky.ru
В строке меню кликнуть "Загрузить", убедиться, что страница с загрузками не открывается.

Блокировка загрузки страницы (HTTPs, SPDY)
Страница: https://google.com
В меню Google кликнуть "Youtube". Убедиться, что страница не загружается.

Блокировка отсылки POST-формы(HTTPS)
Страница: https://mail.ru
Ввести имя пользователя и пароль. Убедиться, что форма содержащая имя пользователя и пароль не была отправлена на сервер.


Вкладка "Get URL"

Загрузка дополнительной страницы в фоне (HTTP, SPDY)
Страница: mail.yahoo.com
При открытии страницы будет сделан фоновый запрос на vk.com с имененм пользователя, сохранённым в
 переменной "account" по условию, указанному, в вкладке "Post Params"
